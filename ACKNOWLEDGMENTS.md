# Acknowledgments

## Inspiration and Attribution

PrivaChat Agents is inspired by and builds upon concepts from **[Perplexica](https://github.com/ItzCrazyKns/Perplexica)** by [ItzCrazyKns](https://github.com/ItzCrazyKns).

We are grateful for the innovative work on AI-powered search and research systems that Perplexica pioneered. While PrivaChat Agents is a complete rewrite with a different architecture (Pydantic AI agents vs. LangChain), we acknowledge the inspiration and ideas that came from studying the Perplexica project.

## Open Source Projects

PrivaChat Agents would not be possible without these excellent open source projects:

### Core Framework
- **[Pydantic AI](https://github.com/pydantic/pydantic-ai)** - Type-safe AI agent framework
- **[FastAPI](https://github.com/tiangolo/fastapi)** - Modern web framework
- **[SQLAlchemy](https://github.com/sqlalchemy/sqlalchemy)** - Database ORM
- **[PostgreSQL](https://www.postgresql.org/)** - Powerful open source database

### AI & ML
- **[OpenRouter](https://openrouter.ai/)** - Unified LLM API gateway
- **[Sentence-Transformers](https://github.com/UKPLab/sentence-transformers)** - Embedding models
- **[pgvector](https://github.com/pgvector/pgvector)** - Vector similarity search
- **[Langfuse](https://github.com/langfuse/langfuse)** - LLM observability

### Document Processing
- **[Dockling](https://github.com/DS4SD/docling)** - PDF, Excel, Word parsing

### Search & Discovery
- **[SearxNG](https://github.com/searxng/searxng)** - Privacy-respecting metasearch engine
